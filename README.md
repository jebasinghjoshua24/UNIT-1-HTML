**Introduction**
        This unit covers the fundamentals of HTML, the standard markup language used to create and structure web pages. It focuses on understanding basic HTML syntax, elements, attributes, and page organization, which form the foundation of web development.

**Objectives**
        After completing this unit, the learner will be able to:
        Understand the structure of an HTML document
        Use common HTML tags and elements correctly
        Create multi-page websites using links and navigation
        Embed media such as images, audio, video, and external conten
        Design basic forms using HTML controls
        Apply semantic HTML for accessibility and clarity

**Topics Covered**

1. HTML Basics
       HTML file structure
        HTML boilerplate
        Headings (h1 to h6)
        Paragraphs and text formatting
        HTML entities and special characters

2. HTML Elements & Attributes
        Tags and elements
        Attributes and values
        Block and inline elements
        Display behavior differences

3. Lists, Links & Navigation
        Ordered, unordered, and description lists
        Anchor tags (<a>)
        Internal and external linking
        Simple navigation menus

4. Media Embedding
        Image embedding (<img>)
        Audio and video embedding (<audio>, <video>)
        Supported media formats
        Iframe embedding (maps and external content)

5. Tables
        Basic table structure
        Table headers, rows, and data
        Rowspan and colspan
        Table accessibility and styling

6. Forms
        Form structure (<form>)
        Input controls (text, email, number, date)
        Buttons, radio buttons, checkboxes
        HTML5 validation attributes
        Field grouping using <fieldset>
        
7. Semantic HTML
        Semantic elements (header, nav, section, article, footer)
        Accessibility basics
        Importance of semantic structure
   
9. Website Structure & Planning
        Folder and file organization
        Linking multiple pages
        Multi-page website planning

Development of Home, About, and Contact pages
