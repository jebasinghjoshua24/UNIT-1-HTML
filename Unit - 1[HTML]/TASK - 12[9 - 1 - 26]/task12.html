Block vs Inline Elements and Layout Examples

Block Elements

Block elements (e.g., <div>, <p>, <h1>) start on a new line, take up the full width available, and stack vertically.

This is a <div> (block element)
Another <div> (block element)

Inline Elements

Inline elements (e.g., <span>, <a>, <strong>) do not start on a new line and only take up as much width as necessary.

This is a <span> (inline) within a sentence, <strong> (inline) and more text follows <a> (inline).

Inline-block Element

display: inline-block; allows elements to flow inline, but have block-like width and height.

Summary

  • Inline elements flow inside a line, do not break the line.
